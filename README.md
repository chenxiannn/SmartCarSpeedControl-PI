# SmartCarSpeedControl-PI
智能车速度PI控制器

飞思卡尔智能车比赛Matlab仿真速度控制PI的模型
Matlab R2011a

- 在Matlab下，打开SpeedControl.mdl,运行即可，点击Scope查看仿真结果
- C程序部分源码在Control文件夹下，主要实现在EIT_PID.c里面
